# Viglet Shio CMS

**Viglet Shio CMS** is an open source solution which allows model content, use GraphQL and create site using Javascript with native cache and search.

- **Current Release:** 0.3.7
- **GitHub:** [https://github.com/ShioCMS](https://github.com/ShioCMS)
