# Viglet Shio CMS

**Viglet Shio CMS** is an open-source headless Content Management System that lets you **model content**, query it via **GraphQL**, and build websites using **JavaScript** with native **Hazelcast caching** and integrated **full-text search** powered by Viglet Turing ES.
