Shio CMS is a headless content management system that lets you model content with custom Post Types, query it via GraphQL and REST API, and build websites using server-side JavaScript with Hazelcast distributed caching and Viglet Turing ES search integration. This document covers architecture, installation, configuration, content modeling, website development, and the developer guide.
