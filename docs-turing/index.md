# Viglet Turing ES

**Viglet Turing ES** is an open-source enterprise search platform with **Semantic Navigation** and **Generative AI**. Index content from any source, expose faceted search experiences, and enable conversational AI grounded in your own content.

- **Current Release:** 2026.1
- **GitHub:** [https://github.com/openviglet/turing](https://github.com/openviglet/turing)
